NetHax

A hacking simulation tool

This program was created to impress friends and family.

To run it, follow these steps:

  • Please install Python 3.x.

  • Then clone the entire git repository or download the zip file and extract it.

  • After that, open Command Prompt or Terminal in that extracted folder or navigate to it using "cd [path]".

  • Now run "python nethax.py" or "python3 nethax.py".

  • Install the displayed modules using "pip install [module]" or "pip3 install [module]".

  • Then repeat step number 4 and it should start.

The project is not completely finished as I still need to find a way to extract wifi data on windows and linux.

I also need to add more functionality to the other options.

If you encounter any errors or have suggestions on how to improve this product,

please contact me or create an issue/pull request on the git repository.
