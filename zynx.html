Zynx - Webbrowser

I created this Webbrowser out of boredom and I'm relatively happy on how it turned out.
It uses the open-source project "Chromium" implemented into python via the module "PyQt5".

If you install PyQt5 and try to run this python script,
it wont work because "PyQt5.QtWebEngineWidgets" does not exist or cannot be found.
To resolve this problem i will provide a download for the working version next to the download for the actual browser. 

If you just download the installer and not worry about the python script, everything should work fine because you don't even need python to run the installed executable.
