Zynx

An open source single-tab browser

I created this webbrowser out of boredom and I'm relatively happy on how it turned out.

It's using the open-source project Chromium implemented into python via the module "PyQt5".

For the python script to work, you need to install the PyQt5 library.

To install it, run pip install PyQt5 PyQt5-sip PyQtWebEngine in your preferred terminal application.

If the browser was installed using the installer, there is no need to install PyQt5 because the executable emulates a python environment with PyQt5 preinstalled.

If you encounter any issues or want to contribute, check out the git repository!

You can check out the startpage right here in your browser!
